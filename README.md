# AI Drama Prompt QA Checklist

English | [简体中文](README.zh-CN.md)

Prompt QA checklists for AI drama production: story intent, character consistency, scene continuity, visual style, and revision notes.

This repository is a neutral, practical resource for AI drama prompt QA, story intent review, character consistency, scene continuity, visual style checks, and revision notes. It is designed to help teams make clearer decisions, write better review notes, and keep workflows reusable.

## Why This Repository Exists

AI drama prompts should be reviewed before they generate production assets. Prompt QA helps teams catch unclear story intent, inconsistent character details, weak scene continuity, and visual style drift early.

## Who This Is For

- AI film teams reviewing prompts before generation
- short drama creators improving prompt consistency
- storyboard teams checking scene and character inputs

## Core Workflow

1. Define the review or production goal
2. Collect the required inputs and constraints
3. Use one template to make decisions comparable
4. Document fit, risks, and next actions
5. Refresh the workflow after real usage data
