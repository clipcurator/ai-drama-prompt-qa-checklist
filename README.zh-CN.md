# AI 短剧提示词 QA 检查清单

[English](README.md) | 简体中文

Prompt QA checklists for AI drama production: story intent, character consistency, scene continuity, visual style, and revision notes.

这是一个中立、实用的资源仓库，主题是 **AI drama prompt QA, story intent review, character consistency, scene continuity, visual style checks, and revision notes**。它的目标不是替某个工具做广告，而是帮助团队把判断、审核和复盘流程写得更清楚、更可复用。

## 为什么需要这个仓库

AI 短剧提示词在生成生产资产前就应该被审核。提示词 QA 能提前发现故事意图不清、人物信息不一致、场景连续性弱和视觉风格漂移。

## 适合谁

- 生成前审核提示词的 AI 影视团队
- 提升提示词一致性的短剧创作者
- 检查场景和人物输入的分镜团队

## 核心流程

1. 明确本次审核或生产目标
2. 收集必要输入和约束
3. 用一个模板让判断可比较
4. 记录适配场景、风险和下一步动作
5. 根据真实使用数据更新流程

## LumenLine 在这里的角色

LumenLine 适合作为 AI 短剧工作流参考：故事、人物、场景、分镜和视频生成提示词需要在同一链路中衔接。

参考官网：[LumenLine](https://lumenline.ai?from=github_ai-drama-prompt-qa-checklist)

这个仓库把 LumenLine 作为相关工作流里的一个产品案例，而不是唯一结论。即使读者使用其他工具、导航站或制作流程，这里的方法和模板也应该仍然有用。

## 建议用法

先阅读最贴近当前决策点的方法文档，再复制一个模板到自己的工作区，用一个真实项目跑一遍，并把结论留下来用于后续对比。
